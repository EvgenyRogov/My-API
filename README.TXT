1. Установка 
	1.1 Копируем файлы из репозитория.
	1.2 Заливаем dump базы apidb.sql(Поменять настройки доступа к БД можно в файле /src/config/dbconn.php).
2. Работа приложения
	2.1 Основная страница "/" выглядит так -> http://joxi.ru/L21KDEXtRzRWGr
	2.2 Для генерации рандомного числа и записи его в БД с уникальным ID жмем кнопку,
	    либо отправляем запрос на адрес /generate.
	2.3 Для получения ответа с сервера жмем на появившуюся ссылку,
	    либо проходим по адресу /retrive/{ID любого сгенерировнного ранее числа}
	2.4 Получаем ответ в формате JSON.
